# 🎬 新电影项目策划报告（投资人路演版）

## 一、项目概述

### 1.1 项目名称
**《记忆重生》**（英文名：*Memory Reborn*）

### 1.2 类型定位
**科幻 / 家庭伦理 / 悬疑**
类型配比：科幻（40%）+ 家庭情感（35%）+ 悬疑犯罪（25%）

### 1.3 核心概念（一句话梗概）
> 近未来中国，记忆数字化存储技术成熟——一位父亲为拯救濒死女儿，将女儿记忆移植到AI仿生体中，却发现"她"记得的真相，远比死亡更可怕。

### 1.4 对标作品

| 对标影片 | 相似维度 | 票房参考 |
|---|---|---|
| 《流浪地球2》 | 中国科幻工业标杆 | 40.29亿 |
| 《人生大事》 | 家庭情感+生死议题 | 17.12亿 |
| 《误杀》系列 | 悬疑反转+亲情驱动 | 13.33亿 |
| 《Her》 | 科技与情感边界探讨 | 全球2.5亿美元 |
| 《降临》 | 科幻外壳+人性内核 | 全球2.03亿美元 |

### 1.5 差异化定位（USP）

1. **"中国式家庭伦理 × 硬科幻设定"**——目前市场空白地带
2. **非宏大叙事，而是"小家庭大议题"**——降低制作门槛，提高情感共鸣
3. **记忆移植的伦理困境**——具备社会话题引爆潜力（类比AI换脸/Deepfake讨论热度）
4. **悬疑反转结构**——满足类型片观众期待，提升口碑传播力

## 二、创意发想过程

### 2.1 第一轮发想：科幻电影新主题创意（3个方向）

**【方向A：SCAMPER-Combine 融合法】**
**《记忆银行》**——将"金融银行系统"与"人类记忆存储"融合。近未来中国，记忆可像货币一样存入"记忆银行"进行交易、借贷、抵押。一个因记忆贷款而失去人生关键记忆的中年男人，必须在记忆彻底清零前找回"抵押品"。

**【方向B：SCAMPER-Adapt 借鉴法】**
**《深海静默》**——借鉴鲸落生态系统的"一鲸落万物生"模式。中国在南海建立深海科研站，发现一种以人类集体潜意识的"精神能量"为食的深海智慧生命体。当科研站失联，一支由科学家和退伍军人组成的救援队发现：这种生命体正在"消化"人类最黑暗的记忆。

**【方向C：SCAMPER-Modify 尺度变换法】**
**《记忆重生》**——将"记忆移植"的尺度从社会层面缩小到一个家庭。近未来中国，记忆数字化技术成熟——一位父亲为拯救濒死女儿，将女儿记忆移植到AI仿生体中，却发现"她"记得的真相远比死亡更可怕。

### 2.2 选择理由：方向C胜出

| 评估维度（权重） | A:记忆银行 | B:深海静默 | C:记忆重生 |
|---|---|---|---|
| 情感共鸣度（30%） | ★★★☆☆ | ★★☆☆☆ | ★★★★★ |
| 制作可行性（25%） | ★★★★☆ | ★★☆☆☆ | ★★★★☆ |
| 市场差异化（20%） | ★★★★☆ | ★★★☆☆ | ★★★★★ |
| 社会话题性（15%） | ★★★★☆ | ★★★☆☆ | ★★★★★ |
| 系列化潜力（10%） | ★★★☆☆ | ★★★☆☆ | ★★★★☆ |
| **加权总分** | **3.55** | **2.45** | **4.65** |

**核心决策逻辑：**
- 方向C以"家庭"为切口，符合中国观众"家文化"情感基因
- 制作成本可控（非太空/深海等重工业场景）
- "记忆移植"是AI时代最直接的伦理议题，具备天然话题性
- 悬疑反转结构提供强类型支撑，降低市场风险

### 2.3 第二轮发想：角色与情节展开（3个方案）

**【方案1：SCAMPER-Combine】**
父亲陈建国（48岁，神经科学教授）——正是他主导了记忆移植技术的研发。女儿小雨（14岁）患罕见脑疾。当妻子因女儿病情崩溃，陈建国在绝望中启动未通过伦理审查的"记忆移植+AI仿生体"实验。**核心张力：父亲既是"上帝"也是"罪人"。**

**【方案2：SCAMPER-Adapt】**
借鉴"忒修斯之船"哲学悖论。女儿林小满（17岁）在车祸后接受记忆移植，但每次记忆"回灌"都会丢失一部分原始记忆，被AI算法"补全"。到第七次回灌时，她已无法确定自己是谁。母亲苏敏（刑警）在调查中发现：女儿车祸并非意外。

**【方案3：SCAMPER-Modify】**
从"被移植者"视角切换为"移植技术执行者"视角。主角周远（35岁，记忆移植中心首席工程师）每天经手数百人的记忆数据。某天，他在一段待删除的记忆数据中，看到了自己失踪十年的妹妹。

### 2.4 最终选择：方案1 + 方案2融合

采用方案1的情感内核（父女关系）+ 方案2的悬疑结构（记忆移植背后的阴谋），形成**"情感驱动+类型支撑"的双引擎结构**。

**最终故事框架：**
- **第一幕**：女儿患病 → 父亲启动非法记忆移植 → "新女儿"诞生
- **第二幕**："新女儿"出现异常记忆碎片 → 指向一桩未解谋杀案
- **第三幕**：父亲发现女儿的记忆中藏着某个真相 → 追查 → 反转
- **高潮**：父亲必须在"删除真相保护女儿"与"揭露真相失去女儿"之间抉择
