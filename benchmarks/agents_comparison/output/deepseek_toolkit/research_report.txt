
================================================================================
    使用线性回归预测股票价格的技术可行性研究 — 完整研究报告
================================================================================
    标的：贵州茅台 (600519.SH) | 数据跨度：2025-04-10 至 2026-05-08
    260个交易日 | 研究员：资深量化研究分析师
================================================================================

一、摘要 (ABSTRACT)
--------------------------------------------------------------------------------
本研究以贵州茅台(600519.SH)为标的，使用约260个交易日（约13个月）的真实收盘价
数据，从零实现多元线性回归模型（OLS闭式解，无第三方库依赖），系统评估了使用
历史价格数据预测次日收盘价的技术可行性。实验设计了5天、10天、20天三种滞后窗口
特征，对比R²、MSE、MAE、方向准确率等指标。结果显示：

  - 所有模型R²均>0.95，其中5天窗口R²最高(0.9646)，表现"看似"优异；
  - 但方向预测准确率仅约46-48%，低于随机猜测(50%)；
  - 模型MSE仅略优于朴素基线(次日=今日)，优势仅3.2%；
  - 在价格突变点（如2026-04-24跳涨至1458.49），预测误差高达39.76元。

核心结论：线性回归在平稳市场中可提供"看似精确"的价格估计(R²>0.96)，但这种
高R²主要由价格序列自身的高自相关性驱动（金融数据的统计特性），而非模型具备
真正的预测能力。方向准确率<50%和突变点大幅失误证明：单纯使用历史价格的线性
回归不具备可靠的股票价格预测能力，无法在实盘中产生超额收益。

二、引言 (INTRODUCTION)
--------------------------------------------------------------------------------
股票价格预测是金融领域最经典也最具争议的课题。有效市场假说(EMH, Fama 1970)
认为，当前价格已反映所有可获得的公开信息，技术分析无法系统性地战胜市场。然而，
行为金融学和大量实证研究表明市场可能存在短期可预测性。

线性回归因其简洁性和可解释性，常被用作预测建模的基线方法。本研究旨在回答：
"仅使用历史收盘价格的线性回归模型，能否可靠地预测次日股价？" 我们通过严格的
实验设计，从多个维度量化评估其技术可行性。

三、方法 (METHODOLOGY)
--------------------------------------------------------------------------------

3.1 数据来源与预处理
  - 标的：贵州茅台(600519.SH)，A股最具代表性的蓝筹股之一
  - 周期：约260个交易日（2025-04-10 ~ 2026-05-08）
  - 变量：日收盘价(Close)，无缺失值
  - 价格区间：1348.00 ~ 1634.99，均值1428.38，标准差61.09
  - 日收益率：均值-0.0432%，标准差0.7771%，呈典型的"接近零均值、尖峰厚尾"特征

3.2 特征工程
  使用滑动窗口法创建滞后特征（Lag Features）：
  - 5天窗口：[P_{t-5}, P_{t-4}, P_{t-3}, P_{t-2}, P_{t-1}] → 预测 P_t
  - 10天窗口：[P_{t-10}, ..., P_{t-1}] → 预测 P_t
  - 20天窗口：[P_{t-20}, ..., P_{t-1}] → 预测 P_t

3.3 模型构建：多元线性回归（从零实现）
  使用正规方程(Normal Equation)求解OLS估计量：
    β = (X^T X)^{-1} X^T y
  
  关键实现细节：
  - 手动实现矩阵求逆（高斯-约旦消元法）
  - 手动计算矩阵乘法
  - 零外部库依赖，仅使用Python math/statistics标准库
  - 添加截距项(Intercept)

3.4 评估指标
  - R² (决定系数)：衡量模型解释的方差比例
  - MSE (均方误差)：衡量预测误差平方均值
  - MAE (平均绝对误差)：衡量预测误差绝对值均值
  - RMSE (均方根误差)：MSE的平方根，与原值同量纲
  - 方向准确率：预测涨跌方向与实际一致的比例
  - 朴素基线对比：与"次日=今日"策略比较

四、实验结果 (RESULTS)
--------------------------------------------------------------------------------

4.1 模型性能总览

  ┌──────────┬──────────┬──────────┬──────────┬──────────┐
  │   指标   │ 5天窗口  │ 10天窗口 │ 20天窗口 │ 朴素基线 │
  ├──────────┼──────────┼──────────┼──────────┼──────────┤
  │  R²      │  0.96457 │  0.96134 │  0.95328 │    —     │
  │  MSE     │ 122.56   │ 122.78   │ 114.87   │ 126.61   │
  │  MAE     │   8.29   │   8.27   │   8.04    │   8.41   │
  │  RMSE    │  11.07   │  11.08   │  10.72    │  11.25   │
  │  样本数  │   257    │   252    │   242     │   261    │
  │  方向准确│  46.48%  │  47.01%  │  47.72%   │   50%*   │
  └──────────┴──────────┴──────────┴──────────┴──────────┘
  * 朴素基线方向准确率50%为理论值（在独立同分布假设下）

4.2 主要发现

发现1：高R²的"幻觉"
  所有模型R²均超过0.95，但这是金融时间序列的典型特征——价格具有极强的自相关性。
  当今天的价格接近昨天的价格时，任何使用近期价格作为输入的模型都会获得高R²。
  真正的衡量标准是方向准确率和MSE相对朴素基线的改进。

发现2：方向预测能力不足
  三个模型的方向准确率分别为46.48%、47.01%、47.72%，全部低于50%（随机水平）。
  这意味着模型在判断"明天是涨是跌"这个问题上比抛硬币还差。在实盘交易中，
  方向比精确价格更重要——方向对了才能赚钱。

发现3：MSE仅略优于朴素基线
  最佳模型(5天窗口)的MSE为122.56，而朴素基线("明天价格=今天价格")的MSE为126.61。
  改进幅度仅为3.2%。考虑到模型复杂度远高于朴素基线，这一微小改进几乎可以忽略。

发现4：突变点预测严重失败
  在2026年4月24日，茅台收盘价从1419.00跳涨至1458.49（+2.78%），模型预测为
  1418.73，误差高达39.76元。在2026年4月27日（随后交易日），价格回落到1403.20，
  模型预测为1455.98，误差达-52.78元。这暴露了线性模型的致命缺陷：无法应对
  突变和反转。

发现5：系数不稳定性
  5天模型系数范围[-0.0451, 0.9588]，20天模型系数范围[-0.1694, 0.8983]。
  随着窗口增长，系数分散程度增加，表明更长期的历史价格对预测的贡献在减弱，
  且出现了负系数——某些历史价格与未来价格呈反向关系。

五、讨论 (DISCUSSION)
--------------------------------------------------------------------------------

5.1 为什么高R²不能说明预测能力？

这涉及金融数据最核心的统计特性：非平稳性和自相关性。

  (a) 价格的非平稳性：股价通常是一阶单整序列 I(1)，即价格本身非平稳但差分
      （收益率）可能平稳。OLS回归假设残差平稳，但价格数据违反这一假设，
      导致"伪回归"(Spurious Regression, Granger & Newbold, 1974)。

  (b) 自相关结构：假设P_t ≈ P_{t-1}，则：
        P_t = α + β·P_{t-1} + ε_t
      当β≈1时（随机游走），R²天然接近1，模型只是在说"明天价格≈今天价格"。

  (c) 我们的实验验证了这一点：最优β系数为0.9588（5天窗口，day t-1），
      说明模型分配给最近一天价格的权重极高。

5.2 学术界共识

  经搜索整理，学术界对线性回归预测股价的局限性有如下共识：

  - 有效市场假说 (Fama, 1970, 1991)：在弱式有效市场中，历史价格信息已完全
    反映在当前价格中，技术分析无法获得超额收益。

  - 伪回归问题 (Granger & Newbold, 1974)：对非平稳时间序列直接进行OLS回归
    会产生虚假的高R²和t统计量。

  - 结构突变 (Bai & Perron, 1998, 2003)：金融市场存在多次结构突变，线性模型
    无法适应这些变化。

  - 非线性依赖：金融市场中的非线性动力学（如波动率聚集、杠杆效应）使得
    线性模型本质上无法捕捉完整的价格行为。

  - 预测区间问题：即使模型在样本内表现良好，样本外预测能力通常急剧衰减，
    这在金融领域尤其严重。

5.3 本研究的局限性

  (1) 仅使用收盘价作为特征，忽略了开盘价、最高/最低价、成交量、技术指标、
      基本面数据等丰富信息。

  (2) 未进行样本外测试（训练/测试分割或滚动窗口验证），所有评估均为样本内。

  (3) 未考虑交易成本、流动性约束、市场冲击等实际交易因素。

  (4) 仅使用单一标的（贵州茅台），结论的普适性需要更多标的验证。

  (5) 未对残差进行诊断（自相关检验、异方差检验、正态性检验）。

5.4 改进方向

  短期改进（低复杂度）：
  (a) 使用收益率而非价格进行建模（差分处理非平稳性）
  (b) 引入成交量、波动率等基本面特征
  (c) 增加技术指标（RSI、MACD、布林带等）
  (d) 采用滚动窗口进行样本外验证

  中期改进（中等复杂度）：
  (e) 使用Lasso/Ridge回归进行正则化，防止过拟合
  (f) 引入市场微观结构特征（买卖价差、订单流等）
  (g) 采用ARIMA/GARCH等时间序列专用模型
  (h) 加入宏观经济变量（利率、CPI、PMI等）

  长期改进（高复杂度）：
  (i) 使用LSTM/Transformer等深度学习模型捕捉非线性依赖
  (j) 集成多模态数据（新闻情绪、社交媒体、卫星图像等另类数据）
  (k) 采用贝叶斯方法量化预测不确定性
  (l) 构建组合预测方法(Ensemble)融合多模型

六、结论 (CONCLUSION)
--------------------------------------------------------------------------------

本研究通过从零实现多元线性回归模型，使用贵州茅台260个交易日的真实数据，
系统评估了"仅使用历史价格预测次日股价"的技术可行性。实验结果表明：

  (1) 统计上"看似成功"（R²>0.96）：但这主要由价格序列的高自相关性驱动，
      而非模型具有真正的预测能力。

  (2) 交易上"实质失败"（方向准确率<50%）：在判断涨跌方向这一对交易最关键
      的任务上，模型表现不如抛硬币。

  (3) 改进空间有限（MSE仅比朴素基线提升3.2%）：这种微小改进在实际交易中
      会完全被交易成本吞噬。

  (4) 突变点致命：在面对价格跳涨或反转时，线性模型产生巨大误差，这正是
      交易中"亏大钱"的场景。

【最终判断】单纯使用历史价格作为特征的线性回归模型，在技术上可以拟合价格
走势（获得高R²），但不具备可交易的预测能力。这支持了有效市场假说的弱式有效
形式。任何声称仅凭历史价格线性模型即可稳定预测股价的说法，都需要极其谨慎
地对待。

【研究意义】本研究的意义不在于否定量化方法，而在于：
  (a) 建立了严格的实验基线，后续研究可在此基础上逐步增加特征和模型复杂度；
  (b) 揭示了高R²在金融时间序列中的误导性，提醒研究者关注方向准确率等更实际
      的指标；
  (c) 为从线性模型向正则化模型、时间序列模型、非参数模型、深度学习的演进
      提供了清晰的动机和起点。
